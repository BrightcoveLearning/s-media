Adding a transcript file

The transcript files can be uploaded using the Media module. To upload a transcript for a video, follow these steps:

  1. Open the Media module.
  2. Click the video title link.
  3. Select the Tracks tab, locate the Transcripts section, click + Add
  4. Select if the transcript file will be uploaded to {{ site.product_short }} or hosted on an external URL.
    • Select files - Select the transcript file from the file system. The file will be uploaded to {{ site.product_short }}.
    • Upload from URL - Enter the public URL where the transcript file resides. The file will be uploaded to {{ site.product_short }}.
  5. Select the Language of the transcript data.
  6. The Label will be added automatically for the transcript file, but you can also edit it as needed.
  7. Select a Status:
    • Publish - Transcript will appear in the video.
    • Draft - Transcript will not appear in the video.
  8. Click Add to save the transcript information.

Editing transcript details

The Media module provides the option to edit transcript details. To edit transcript details, follow these steps.

  1. Open the Media module.
  2. Click the link for the video you want to edit.
  3. At the top of the page, select the Tracks tab.
  4. Locate the Transcripts section and click on and select the Edit transcript details option.
  5. Edit the transcript details as needed.
  6. Click Save.

Downloading transcript files

The Media module provides the ability to download transcript files. To download a transcript file, follow these steps.

  1. Open the Media module.
  2. Click the link for the video you want to edit.
  3. At the top of the page, select the Tracks tab.
  4. Locate the Transcripts section and click and select the Download option.

Deleting transcript files

The Media module provides the option to delete transcripts. To delete transcripts, follow these steps.

  1. Open the Media module.
  2. Click the link for the video you want to edit.
  3. At the top of the page, select the Tracks tab.
  4. Locate the Transcripts section and click on and select the Delete option

Limitations

  • Formats other than .txt or JSON on upload are not supported.
  • If you are using Auto Captioning, the transcript will show, but this will be in a JSON structure with auto transcription errors. Note that there is no way to correct these errors without downloading, fixing, and uploading the transcripts again.
  • No Brightcove Player plugin currently shows the transcript and enables player control/interaction.
  • Remote Transcripts are not initially supported.
  • Transcript upload on shared assets is not supported.
