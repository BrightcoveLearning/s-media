Overview

The updated search experience in Video Cloud Studio brings significant improvements to how you can search for and manage your video content. Key updates include a redesigned UI, advanced search filters, and enhanced sorting options.

Layout

The updated user interface for Video Cloud search includes several significant changes aimed at improving usability and efficiency:

Layout updates

  • Tabs for Search: Switch between Videos Search and Playlist Search using dedicated tabs.
  • Search Bar and Add Button: The search bar has been moved to the left for better accessibility, while Add videos has been relocated to the right, streamlining the overall layout.

    By clicking the Add videos button, you can perform various actions to enhance your video management experience. The available actions include:

    • Upload: Allows you to upload new videos from your local storage.

    • Add remote video: Enables you to add videos from remote locations or URLs and access the Video Editor to create new videos.

    • Screen and Camera Recorder: Provides options to record your screen or camera directly and upload the recording.

Content Management: List and Card Views

Our platform now includes two distinct views for exploring and managing your resources:

  • List View: Displays your resources in a detailed table format, providing a comprehensive view of information.
  • Card View: Shows your resources in a card format, providing a more visual and organized layout.

List View summary

  • Sort Fields: Sort results by selecting various columns to display the most relevant information for you.

Card View summary

  • Sort Fields: Sort results by Date Created, Date Modified, or Video Name in both ascending and descending order.
  • Larger Thumbnails: Thumbnails are now larger, making it easier to identify videos at a glance.
  • Key Information on Hover: Hovering over a video thumbnail reveals key information, such as ID and Tags.

These updates aim to enhance the user experience, making it more intuitive and efficient to search and manage video content within the Video Cloud Studio.

Search

The updated search functionality in Video Cloud includes several advanced filtering options to refine search results effectively.

  • Partial Matches: Results match any part of the entered string in Name or ID.

    • Example: Typing est will return videos with test or festival in their names.

  • Keyboard Trigger: Searches initiate upon keyboard entry after the 3rd character.

Search Filters

The following search filters can be applied individually or together to refine search results:

  • String: Text-based search.

  • List: Choose from predefined lists.

  • Tag: Filter by specific tags.

  • Date: Filter by date.

  • Boolean: True/False conditions.

  • Duration: Filter by video duration.

Logic and Appearance

  • AND Logic: The logic between each filter is an AND operation, meaning all conditions must be met.

  • Color Coding: Excludes are shown in a different color/state.

  • Quick Identification: As the number of filters grow, a search to quickly identify the Search Options is available.

Working with Results

The updated search experience in Video Cloud provides several new functionalities for managing search results effectively:

  • Checkbox Selection: Selecting a checkbox now presents a new component at the bottom of the results for batch actions.

  • Single Preview and Metadata Edit: By clicking you can preview and edit metadata for individual videos directly from the search results.

  • Bulk Metadata Selection: Allows batch editing of metadata for multiple videos at once.

  • Export Functionality: The export button continues to work as before, enabling you to export search results with your metadata.

For more advanced search options, see the CMS API: Video Search v2 documentation.
