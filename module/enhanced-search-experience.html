Overview

The new search experience in Video Cloud Studio brings significant improvements to how users search for and manage their video content. Key updates include a redesigned UI, advanced search filters, and enhanced sorting options.

UI-Based Updates

The updated user interface for Video Cloud search includes several significant changes aimed at improving usability and efficiency:

Layout Changes

  • Tabs for Search: Switch between Videos Search and Playlist Search using dedicated tabs.
  • Search Bar and Add Button: The search bar has been moved to the left for better accessibility, while Add has been relocated to the right, streamlining the overall layout.

Card View of Results

  • Sort Fields: Users can sort results by Created, Modified, or Name in both ascending and descending order.
  • Larger Thumbnails: Thumbnails are now larger, making it easier to identify videos at a glance.
  • Key Information on Hover: Hovering over a video thumbnail reveals key information, such as Date, ID and Tags.

Table View of Results

  • Metadata Column Selection: Users can choose which metadata columns to display, customizing their view to suit their needs.
  • Sortable Column Headers: Sorting is available directly from the column headers, allowing for quick reorganization of the displayed data.

These updates aim to enhance the user experience, making it more intuitive and efficient to search and manage video content within the Video Cloud Studio.

Search Filters

The updated search functionality in Video Cloud includes several advanced filtering options to refine search results effectively.

Partial String Matching

  • Partial Matches: Results match any part of the entered string in Name or ID.

    • Example: Typing est will return videos with test or festival in their names.

  • Keyboard Trigger: Searches initiate upon keyboard entry.

Search Filters

Adding filters helps refine the search. The following search filters are supported:

  • String: Text-based search.

  • List: Choose from predefined lists.

  • Tag: Filter by specific tags.

  • Date: True/False conditions.

  • Boolean: Text-based search.

  • Duration: Filter by video duration.

Logic and Appearance

  • AND Logic: The logic between each filter is an AND operation, meaning all conditions must be met.

  • Color Coding: excludes are shown in a different color/state.

  • Quick Identification: As the number of filters grow, a search to quickly identify the filter is available.

Working with Results

The updated search experience in Video Cloud provides several new functionalities for managing search results effectively:

  • Checkbox Selection: Selecting a checkbox now presents a new component at the bottom of the results for batch actions.

  • Single Preview and Metadata Edit: Users can preview and edit metadata for individual videos directly from the search results.

  • Bulk Metadata Selection: Allows batch editing of metadata for multiple videos at once.

  • Export Functionality: The export button continues to work as before, enabling users to export search results with their metadata.
